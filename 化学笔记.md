## 空气

| 物质     | $N_2$                | $O_2$               | $CO_2$ | 稀有气体                                       | 其他气体 |
| -------- | -------------------- | ------------------- | ------ | ---------------------------------------------- | -------- |
| 占比     | 78%                  | 21%                 | 0.03%  | 0.94%                                          | 0.03%    |
| 物理性质 | 常温下为无色无味气体 |                     |        | 通电会发出五颜六色的光                         |          |
| 化学性质 | 不活泼，惰性         | 相对于 $N_2$ 更活泼 |        | 相对于 $N_2$ 更稳定                            |          |
| 用途     | 保护气，食品防腐     |                     |        | 电焊，霓虹灯（通电发出各种颜色的光，物理变化） |          |

### 空气中含氧量的测定

1. 拉瓦西
   - 原理： $\ce{Hg + O2 ->[加热] HgO}$
   - $\ce{Hg}$ 是银白色液体， $\ce{HgO}$ 是红色固体。
   - 现象：钟罩内水银升高，有红色固体生成。
   - 结论：
     - 空气中氧气约占其体积的 $ \frac{1}{5} $。
     - 空气中有氧气。
     - 空气由两种或以上气体组成。
     - 空气是混合物。

2. 实验室

   - 原理：$ \ce{P + O2 ->[点燃] P2O5} $

   - 现象：产生大量白烟，放热，发光

   - 检查容器气密性

     1. 将导管放入水中
     2. 双手紧握集气瓶，观察到导管口有气泡冒出
     3. 松开手，若导管内有稳定水柱，则气密性良好

   - 将红磷点燃，迅速放到瓶中。关好瓶盖。

   - 反应结束，等待其冷却至室温。打开止水夹，水被倒吸入集气瓶，且体积占集气瓶空气体积的 $ \frac{1}{5} $。

   - 结论：氧气占空气体积的 $ \frac{1}{5} $。

   - 误差：

     | 错误           | 结果 |
     | -------------- | ---- |
     | 气密性不良     | 偏小 |
     | 插入速度过慢   | 偏大 |
     | 没有冷却到室温 | 偏小 |
     | 止水夹未夹紧   | 偏大 |
     | 红磷不足       | 偏小 |
   
   - 优化改变：
     - 在集气瓶中加水，吸收 $\ce{P2O5}$，加速冷却
   
   - 替代品：
     - 只与氧气反应
     - 反应物为固体或液体
     - 生成物为固体或液体
   
   - 空气污染物：
   
     - $\ce{CO}$
     - 酸雨：
       - $\ce{SO2}$
       - $\ce{NO2}$
     - $\ce{O3}$
     - 可吸入颗粒物
       - $\ce{PM_{2.5}}$
       - $\ce{PM_10}$

## 氧气

$\ce{C + O2 = CO2}$ 

检验 $\ce{CO2}$：

- 氢氧化钙+二氧化碳 $\ce{->}$ 碳酸钙+水
- $\ce{Ca(OH)2+CO2 -> CaCO3 + H2O}$

### 氧气的物理性质

在常温常压下：

- 无色无味气体
- 密度大于空气

并且：

- 不易溶于水
- $\ce{101Kp}$ + $-183\degree$ 液化为淡蓝色液体
- $\ce{101Kp}$ + $-218\degree$ 固化为蓝色固体

### 氧气的化学性质

- 助燃性
- $ \ce{C+O2 ->[点燃] CO2}$
  - 燃烧发白光
- $\ce{S+O2->[点燃]SO2}$
  - 燃烧 发淡蓝色火焰
- $\ce{Fe+O2->[点燃]Fe3O4}$

### 铁丝燃烧实验

为什么下面要放水？吸收有害物质 防止污染空气

为什么下面要放水/细沙？防止高温物质溅落使容器破碎

铁丝为什么要螺旋形？增大受热面积/增大与氧气的接触面积

火柴将燃尽时放入瓶体：防止火柴消耗掉大部分氧气

现象：剧烈燃烧，火星四射，放热，生成（高温熔融的）黑色固体

### 氧化反应

（初中）：与氧气的反应。

缓慢氧化：

- 呼吸作用
- 铜粉变绿
- 铁生锈

### 氧气工业制法

分离液态空气法

原料：空气

空气 $\ce{->[加压降温]}$ 液态空气 $\ce{->[升温]}$ 氢气（$\ce{N2}$，196%）+液氧（183%）

液氧是混合物（氧气，稀有气体，二氧化碳，水蒸气等其他气体）

### 实验室制氧气

原料/药品/试剂：双氧水（过氧化氢溶液）$\ce{H2O2}$，$\ce{MnO2}$（催化剂）

催化剂：

- 只改变反应速率，不改变产率
- 化学反应前后，（催化剂）物质的质量和化学性质不变。

双氧水 $\ce{->[MnO2]}$水+氧气

$\ce{H2O2->[MnO2]H2O + O2}$

条件：锥形瓶（反应装置），长颈漏斗

**向上排空气法**：导管深入瓶底，毛玻璃片覆盖瓶口（粗糙面向下）

步骤：

1. 连接装置
2. 检查气密性：夹紧止水夹，向漏斗中加水，没过漏斗口，继续加水。若漏斗导管内有一段稳定水柱，则气密性良好。
3. 装药：双氧水没过漏斗口，若未没过，则导管无法接收到气体。

验满/检验氧气：在集气瓶中放一个带火星的木条，若复燃则集满/生成物为氧气

**收集法2：排水法**（氧气不易溶于水）

优点：收集气体更纯

开始收集的时间：等导管口有连续气泡冒出，气泡冒出时才开始收集（否则不纯）

集满：集气瓶中有大气泡冒出时则集满

在水下用毛玻璃片盖住瓶口，拿出水面，正放在桌面上

#### 优化与改变

将长颈漏斗换为分液漏斗

检查气密性：夹紧止水夹，向漏斗内加入少量的水。打开活塞。若水不滴下则气密性良好

优点：可控制滴落速度和双氧水的量，控制反应速度，使反应平稳进行。得到平稳气流。方便氧气收集。

原案优点：能随时添加液体。

导管口有均匀气泡冒出时开始收集。

### 实验室制取氧气（2）

#### 氯化钾

原理：

氯酸钾 $\ce{->[二氧化锰]}$ 氯化钾+氧气

$\ce{KClO3 ->[MnO2] KCl + O2}$

试管略微朝下倾斜，防止冷凝水倒流使试管炸裂。

#### 高锰酸钾

高锰酸钾 $\ce{->[加热]}$ 锰酸钾 + 二氧化锰 + 氧气

$\ce{KMnO4->[\triangle] K2MnO4 + MnO2 + O2}$

需要玻璃棉！防止高锰酸钾粉末进入导管使导管堵塞

用排水法时注意：先将导管移出水面再熄灭酒精灯。

## 二氧化碳的制备

#### 工业

碳酸钙 是 石灰石，$\ce{CaO}$ 是 生石灰

$ \ce{CaCO3 ->[高温] CaO + CO2} $

碳酸钙 $\ce{->[高温]}$ 氧化钙 + 二氧化碳

#### 实验室

大理石/石灰石 + 稀盐酸 $\ce{->}$ 氯化钙 + 二氧化碳 + 水

$\ce{CaCO3 + HCl -> CaCl2 + CO2 + H2O}$

验满：燃着的木棒

将燃着的木棒放到集气瓶中，若熄灭，则满

检验：向集气瓶中和加澄清石灰水，若澄清石灰水变浑浊，则该气体为二氧化碳

用石灰石/大理石（主要成分：碳酸钙）而不用碳酸钙粉末（反应过快）。

## 水的性质

宏观：肉眼可见

微观：肉眼不可见

水的微观形式：无数水分子凝聚而成

水分子体积和质量都很小

水的三态：改变分子之间的间隔，排列方式

不改变：分子种类，质量，体积

水是由分子构成的物质。分子是保持其化学性质的最小微粒。

### 分子

1. 体积，质量极小
2. 分子之间有间隔
3. 分子在不断运动（分子运动越快，温度越高）

### 水的物理性质

- 无色无味液体
- 反常膨胀
- 表面有张力
- 有毛细现象
- 比热容大
- 有良好的溶解其他物质的能力（是良好的溶剂）

### 水的化学性质

- $\ce{H2O + CO2 -> HaCO3}$
- $\ce{HaCO3 ->[\Delta] H2O + CO2}$
- $\ce{H2O + CaO -> Ca(OH)2}$ 放出大量热
- $\ce{H2O + CaSO4 -> CaSO4 · 5H2O}$

### 其他

酚酞遇碱性物质变红

紫色石蕊遇碱性物质变蓝

紫色石蕊遇酸性物质变红

### 压强变化

1. 温度变化
2. 有气体或液体生成
3. 有气体/液体注入

除水最后，检验水最先

### 水的自然净化

沉降 氧化 吸附 挥发 稀释 分解 微生物

含有杂质的水通过自然过程，使水中污染物浓度减小，使水恢复到污染前的状态

水质好坏判断：溶解氧（重要指标）溶解的氧越多，水质越好

### 水的人工净化

沉降 吸附 过滤 蒸馏

活性炭：吸附 过滤 吸附异味 色素 等 结构疏松多孔

只除去不可溶物质

还有可溶性物质（$\ce{Ca, Mg}$等）

杂质较多：硬水，较少：软水

区分软水：

泡沫水

- 气泡多 浮渣少 软水
- 气泡少 浮渣多 硬水

硬水软化：

- 实验室 蒸馏
- 日常 煮沸

## 分子与原子

物质由分子组成

分子由原子组成

- **原子**：金属单质，稀有气体，非金属单质
- **离子**：由金属或铵根离子组成

- **物理变化**：分子间间隔、排列方式改变 分子种类 质量 数量不变
- **化学变化**：分子分裂为原子，原子组合成新的分子，变分子种类，原子不变

### 分子原子

#### 共同点

1. 体积质量很小
2. 不停运动
3. 有间隔
4. 同种微粒化学性质相同

#### 不同点

1. 由分子组成的物质 分子是保持其化学性质的最小微粒
2. 由原子组成的物质 原子是保持其化学性质的最小微粒

## 原子结构

1. **道尔顿** 认为原子是不可再分的实心球体

2. **汤姆森** 发现带负电的电子 葡萄干模型

3. **卢瑟福** 行星模型：

   - 原子是空心的

   - 由原子核和电子组成

     - 原子质量集中在原子核上

     - 电子被叫做核外电子
       - 一个电子带一个单位负电
     - 原子核由质子和中子组成
       - 一个质子带一个单位正电荷
       - 中子不带电
       - 质子质量约等于中子质量
       - 原子核中的电子等于核外电子数
         - 核电荷数 = 原子系数
